### Bridge II — The Nash-to-Pareto Transition as the Landau-Levich Cooperative Film

In the space of two-player coordination games, the payoff matrix is parameterized by four values: (R, S, T, P) = (Reward for mutual cooperation, Sucker payoff, Temptation to defect, Punishment for mutual defection). The Nash equilibrium (mutual defection, payoff P each) and the Pareto optimum (mutual cooperation, payoff R each) are separated by the **cooperation surplus**:

```

```

Under False-Uniqueness (each player believes their preference for R is unique), the perceived payoff of unilateral cooperation is S (being the only cooperator is catastrophic). The agent plays Nash because they believe P(partner cooperates) ≈ 0.

**The Payoff Transparency Field.** Define the **transparency field** τ ∈ [0, 1] as the probability that each agent correctly knows their partner's true preference (σⱼ = cooperate). Under full transparency (τ = 1), P(partner cooperates | I cooperate) = P(partner cooperates) = true_p_coop, and the dominant strategy shifts to cooperation when true_p_coop > (P − S)/(R + P − T − S). Under MUTE opacity (τ = 0), each agent uses only their manufactured estimate p̂_coop ≈ 0 (False-Uniqueness), and the Prisoner's Dilemma trap is stable.

**The ECHO Capillary Number:**

```
Ca_ECHO  =  (P − S) / (R − P) · (1 − τ)  =  Nash_cost / Cooperation_surplus · opacity
```

Ca_ECHO < 1: transparency sufficient to reach Pareto; cooperation is the dominant strategy; False-Uniqueness dissolved; ECHO regime.
Ca_ECHO > 1: opacity sufficient to maintain Nash trap; cooperation is irrational under perceived payoffs; MUTE regime.
Ca_ECHO = 1: **Pareto Critical Point** — the knife-edge at which a single credible revelation of shared preferences (Abilene revelation, whistle-blower, pre-mortem) tips the system from Nash to Pareto.

**The LLD Cooperative Film.** The Nash-to-Pareto transition follows the Landau-Levich thin-film scaling. Define the **cooperation film thickness** τ* — the minimum transparency required to make cooperation the dominant strategy for an agent with expected partner-cooperation probability p̂_coop:

```
τ*  ~  (Nash_cost / Total_surplus)^{2/3}  =  ((P − S)/(R + P − T − S))^{2/3}
```

This is the LLD scaling: the transparency threshold grows as the two-thirds power of the cost-surplus ratio. Low cooperation surplus (R ≈ P) requires near-total transparency (τ* → 1) to unlock the Pareto state — like a high-capillary-number fluid that only thins at extreme speeds. High cooperation surplus (R >> P) requires minimal transparency (τ* → 0) — like a low-viscosity fluid that coats easily.

**Ergodicity and the Cooperation Film.** Under the ergodicity framework, the Nash equilibrium is the **non-ergodic trap**: in a repeated game with multiplicative payoffs, a player who cooperates unilaterally faces absorbing ruin (S < P on every round). The Pareto optimum is the **ergodic restoration**: mutual cooperation produces a time-average growth rate (μ − σ²/2) > 0 for both players. Peters' Kelly-criterion analogue for the social game:

```
g_coop*  =  μ_coop − σ²_coop / 2  =  R − (R − S)² / (2 · (R − P))
g_Nash   =  μ_Nash − σ²_Nash / 2  =  P − (P − S)² / (2 · (R − P))
```

The **ergodic cooperation gain** Δg_erg = g_coop* − g_Nash > 0 when R > P + (R-S)²/(2(R-P)), which holds for all standard Prisoner's Dilemma parameters. Cooperation is ergodically superior to defection — the time-average of cooperation exceeds the time-average of Nash even for purely self-interested agents — but only when the partnership is stable enough that individual agents are not ruined before the ergodic limit is reached. ECHO restores ergodicity by protecting agents from unilateral-cooperation ruin through the transparency condition τ > τ*.

**Physical template: Landau-Levich cooperative film.** The cooperation film thickness τ* ~ (Nash_cost/Total_surplus)^{2/3} is the ECHO analog of the LLD film thickness h₀ ~ Ca^{2/3}.
